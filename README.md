# 🖥️ УчимсяВместе — Веб-ресурс для повышения компьютерной грамотности

Дипломный проект. Веб-сайт для обучения пожилых людей компьютерной грамотности.

---

## 🛠️ Стек технологий

- **Backend:** Python 3.10+ / Flask
- **База данных:** MySQL (Railway) / SQLite (локально)
- **ORM:** Flask-SQLAlchemy
- **Frontend:** HTML5, CSS3, JavaScript
- **Хостинг:** Railway

---

## 🚀 Быстрый запуск (локально)

### 1. Клонируй репозиторий
```bash
git clone https://github.com/ds23432c/uhakova_site.git
cd uhakova_site
```

### 2. Создай виртуальное окружение
```bash
python -m venv venv
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate
```

### 3. Установи зависимости
```bash
pip install -r requirements.txt
```

### 4. Создай файл .env
```env
DATABASE_URL=mysql+pymysql://root:ПАРОЛЬ@HOST:PORT/railway
SECRET_KEY=любой-секретный-ключ
DEBUG=True
```
> Для локального запуска без MySQL можно не создавать .env — автоматически используется SQLite.

### 5. Запусти приложение
```bash
python app.py
```

Открой браузер: **http://localhost:5000**

---

## 🗄️ База данных

При **первом запуске** автоматически:
- Создаются все таблицы
- Заполняются начальные данные (модули, уроки, тесты, FAQ, словарь, блог)
- Создаётся аккаунт администратора

**При повторных запусках** — БД не трогается!

## 🌐 Деплой на Railway

1. Загрузи код на GitHub
2. Подключи Railway к репозиторию
3. Добавь переменную `DATABASE_URL` с MySQL URL из Railway
4. Добавь `SECRET_KEY`
5. Деплой запустится автоматически!
